# 导航菜单

站点通用头部组件，包含品牌/登录入口、导航区以及用户下拉菜单。默认固定在页面顶部并带边框与高度控制变量。

## 快速上手

```tsx
import { HdHeader } from '@hd/react/components'

function AppLayout({ children }: { children: React.ReactNode }) {
  const menu = (
    <nav className="flex items-center gap-3">
      {/* 这里放你的导航链接 */}
      <a href="/courses" className="text-sm">
        课程
      </a>
      <a href="/projects" className="text-sm">
        项目
      </a>
    </nav>
  )

  return (
    <div className="min-h-dvh flex flex-col">
      <HdHeader
        login={<img src="/logo.svg" alt="logo" className="size-6" />}
        menu={menu}
        userDropdownMenus={{
          label: '快捷入口',
          items: [
            { title: '我的资料', to: '/member?system=profile' },
            { title: '帐号绑定', to: '/member?system=bind' },
          ],
        }}
      />
      <main className="flex-1">{children}</main>
    </div>
  )
}
```

## 功能与结构

- 左侧品牌/登录：`login` 未传时显示默认图标并链接首页
- 导航区：`menu` 节点在桌面端显示（容器自带 `hidden lg:flex`）
- 右侧用户下拉：集成 `UserDropdown`，支持未登录时显示“登录/注册”按钮
- 固定顶部：`sticky top-0 z-10`，高度由 `--header-height` 控制

## Props

| 属性名            | 类型               | 必填 | 说明                              |
| ----------------- | ------------------ | ---: | --------------------------------- |
| login             | ReactNode          |   否 | 左侧品牌/登录入口（覆盖默认图标） |
| menu              | ReactNode          |   是 | 导航区内容（建议为一组链接/菜单） |
| userDropdownMenus | IUserDropdownMenus |   否 | 用户下拉菜单的额外入口配置        |

## 设计约定与定制

- 高度变量：通过 CSS 变量 `--header-height` 控制整体高度
  ```css
  :root {
    --header-height: 64px;
  }
  ```
- 响应式导航：默认只在 `lg` 及以上显示导航；移动端可结合抽屉/侧栏实现
- 用户菜单：`userDropdownMenus.items` 为附加入口，内置“资料管理/帐号绑定”等固定项
