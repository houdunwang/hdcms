# 登录注册使用

该组件用于统一渲染登录、注册、找回密码、微信扫码登录四类认证页面，并通过查询参数 `action` 在同一路由下进行切换。

## 快速上手

- 路由挂载：
  ```tsx
  import { AuthLayout } from '@core/auth/AuthLayout'
  
  export function AuthPage() {
    return (
      <AuthLayout />
    )
  }
  ```
- 页面切换（通过查询参数）：
  - `/auth?action=login`
  - `/auth?action=register`
  - `/auth?action=findPassword`
  - `/auth?action=wechatLogin`

- 常见使用场景：
  - 注入右侧说明区域：`components[action].rightSpace`
  - 开启微信登录入口：`showWechatLoginButton`
  - 自定义微信按钮：`wechatLoginButton`
  - 完全替换页脚：`footerComponent`
  - 增加卡片底部辅助说明：`helperComponent`

## Props（主组件）

| 属性名 | 类型 | 默认值 | 说明 |
|---|---|---:|---|
| className | string | - | 布局容器额外类名 |
| showWechatLoginButton | boolean | false | 是否在页脚显示“微信登录”入口 |
| wechatLoginButton | ReactNode | - | 自定义“微信登录”按钮节点（替换默认按钮样式与文案） |
| footerComponent | ReactNode | - | 完全覆盖默认页脚（含登录/注册/找回密码导航） |
| helperComponent | ReactNode | - | 卡片底部辅助说明区块；存在时卡片底部显示分割线 |
| components | { login?; register?; findPassword?; wechatLogin? } | - | 为各子页面提供配置（见下表） |

## Props（子页面配置 components.*）

| 配置路径 | 类型 | 默认值 | 说明 |
|---|---|---:|---|
| components.login.rightSpace | ReactNode | - | 登录页右侧插槽内容 |
| components.register.rightSpace | ReactNode | - | 注册页右侧插槽内容 |
| components.findPassword.rightSpace | ReactNode | - | 找回密码页右侧插槽内容 |
| components.wechatLogin.rightSpace | ReactNode | - | 微信登录页右侧插槽内容 |
| components.*.title | string | - | 预留字段，当前内置页面使用各自标题 |
| components.*.description | string | - | 预留字段，当前内置页面使用各自描述 |

## 示例（结合右侧插槽与微信入口）

```tsx
import { AuthLayout } from '@core/auth/AuthLayout'
import { LoginRightSpace } from '@core/auth/Login'
import { RegisterRightSpace } from '@core/auth/Register'
import { FindPasswordRightSpace } from '@core/auth/FindPassword'
import { Button } from '@/components/ui/button'

export function AuthPage() {
  return (
    <AuthLayout
      showWechatLoginButton
      wechatLoginButton={<Button variant="default" size="lg" className="w-full">使用微信快速登录</Button>}
      helperComponent={<div className="text-xs">为保障账号安全，系统会记录登录设备与时间。</div>}
      components={{
        login: { rightSpace: <LoginRightSpace /> },
        register: { rightSpace: <RegisterRightSpace /> },
        findPassword: { rightSpace: <FindPasswordRightSpace /> },
      }}
    />
  )
}
```

## 使用约定

- 统一使用 `/auth?action=...` 切换子页面，避免多个独立路由
- 优先通过 `rightSpace` 注入右侧内容，保持主表单区域简洁
- 若覆盖页脚，请保留必要入口（登录/注册/找回密码）以保证用户可达性
