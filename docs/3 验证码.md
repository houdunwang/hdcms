# 手机/邮箱验证码发送

用于发送邮箱或手机号的验证码按钮，内置格式校验、发送状态与倒计时，避免重复点击。

## Props

| 名称 | 类型 | 必填 | 默认值 | 描述 |
| --- | --- | --- | --- | --- |
| type | `'email' \| 'mobile'` | 是 | 无 | 发送验证码的目标类型：邮箱或手机号 |
| value | `string` | 是 | 无 | 与类型匹配的输入值（邮箱地址或手机号） |

## 使用示例

```tsx
import { SendCodeButton } from "@/core/form/SendCodeButton"

function Example() {
  const [email, setEmail] = useState("")
  const [mobile, setMobile] = useState("")

  return (
    <div>
      {/* 邮箱验证码 */}
      <input
        value={email}
        onChange={(e) => setEmail(e.target.value)}
        placeholder="请输入邮箱"
      />
      <SendCodeButton type="email" value={email} />

      {/* 手机验证码 */}
      <input
        value={mobile}
        onChange={(e) => setMobile(e.target.value)}
        placeholder="请输入手机号"
      />
      <SendCodeButton type="mobile" value={mobile} />
    </div>
  )
}
```

## 行为说明

- 当输入为空或格式不正确时，按钮会提示错误并不会发起请求。
- 发送成功后进入 60 秒倒计时，倒计时期间按钮禁用并显示剩余秒数。
- 组件会在本地存储记录剩余时间用于跨刷新保持倒计时。
