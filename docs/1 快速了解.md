## 前端共享库

@hdcms/react 为项目公共前端库，包括丰富的功能，为你开发节省大量时间。

该库是独立的package，不与你的前端项目耦合，所以更新了这个包后你可以享受到更多新功能，不影响你的项目开发。

### 安装配置

app/front 项目已经配置好了，如果你有其他项目想使用可以按照下面方法配置

在 package.json 中添加

```
"dependencies": {
  "@hdcms/react": "workspace:*",
  ...
}
```

然后在你的 shadcn-ui 的css 文件中设置以下规则

```
@source "../../../../packages/react";
@import '../../../../packages/react/src/assets/global.css';
...
```
