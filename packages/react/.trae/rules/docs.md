# 技能：生成使用文档

## 技能描述

此技能用于解析 AdonisJS 控制器文件，并根据其中的 JSDoc 注释和路由定义，自动生成详细的 Markdown 使用文档。

## 生成文档要求

1. 如果是组件，使用表格详细说明props的类型、默认值、是否必填、描述等。

2. 如果是hook，使用表格详细说明返回值的类型、默认值、描述等。

3. 文档是给前端使用者看的，应清晰易懂，避免深入具体使用细节

4. 用中文回复

5. 以 markdown 格式输出

6. 以该文件的名称作为文档的文件名

7. 在 docs 目录中，以文件所在目录名作为文档的目录名，将文件保存到该目录中

8. 不用介绍源代码，只介绍使用方法和参数的详细说明
