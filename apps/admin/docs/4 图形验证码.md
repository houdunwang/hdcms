# 图形验证码

本文档介绍如何在您的应用中使用图形验证码功能。

## 功能概述

图形验证码用于防止机器人自动提交表单，通过要求用户输入图片中显示的字符来验证用户是人类。

## API 端点

通过以下 API 端点可以获取一个新的图形验证码。

- **URL:** `/core/captcha`
- **Method:** `GET`
- **说明:** 此接口用于获取一个 SVG 格式的图形验证码和其唯一标识 `key`。

### 响应

调用成功后，API 会返回一个 JSON 对象，包含用于显示验证码和后续验证所需的数据。

- **状态码:** `200 OK`
- **响应体:**

```json
{
  "key": "string",
  "svg": "string"
}
```

- **字段说明:**
  - `key` (string): 验证码的唯一标识符。在提交表单进行验证时，需要将此 `key` 和用户输入的验证码文本一起发送到后端。此 `key` 的有效期为1分钟。
  - `svg` (string): SVG 格式的验证码图像数据。您可以直接在 HTML 中显示此 SVG。

## 使用示例

前端可以通过直接请求此端点来获取和显示验证码。

### React (TanStack Query)

如果您在 React 项目中使用 TanStack Query，可以像下面这样获取和管理验证码状态。

```jsx
import { useQuery } from '@tanstack/react-query'
import { useState } from 'react'

// 定义一个用于获取验证码的异步函数
const fetchCaptcha = async () => {
  const response = await fetch('/core/captcha')
  if (!response.ok) {
    throw new Error('Failed to fetch captcha')
  }
  return response.json()
}

function CaptchaComponent() {
  const [userInput, setUserInput] = useState('')

  // 使用 useQuery 来获取和缓存验证码数据
  const { data, error, isLoading, isError, refetch } = useQuery({
    queryKey: ['captcha'],
    queryFn: fetchCaptcha,
    staleTime: 60 * 1000, // 与后端缓存时间保持一致（1分钟）
    gcTime: 60 * 1000,
    refetchOnWindowFocus: false, // 避免窗口聚焦时自动刷新
  })

  const handleSubmit = () => {
    console.log('将使用以下信息进行验证:')
    console.log('Key:', data?.key)
    console.log('用户输入:', userInput)
    // 在这里，您需要将 data.key 和 userInput 发送到您的后端进行验证
  }

  if (isLoading) {
    return <div>正在加载验证码...</div>
  }

  if (isError) {
    return <div>验证码加载失败: {error.message}</div>
  }

  return (
    <div>
      <div id="captcha-container" dangerouslySetInnerHTML={{ __html: data.svg }} />
      <input
        type="text"
        value={userInput}
        onChange={(e) => setUserInput(e.target.value)}
        placeholder="请输入验证码"
      />
      <button onClick={handleSubmit}>验证</button>
      <button onClick={() => refetch()}>刷新验证码</button>
    </div>
  )
}

export default CaptchaComponent
```

当用户提交表单时，您需要将 `data.key` 和用户输入的验证码文本一起发送到服务器进行验证。

## 后端实现细节

- **验证码生成:** 使用 `svg-captcha` 库生成。
  - 长度: 3个字符
  - 忽略字符: `0o1i` (排除了容易混淆的字符)
  - 干扰线: 3条
- **缓存:** 验证码的文本内容存储在缓存中，其 `key` 即为 API 返回的 `captchaKey`，有效期为 **20分钟**。
