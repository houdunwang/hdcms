# 登录与注册

本文档介绍如何在您的应用中集成用户登录和注册功能。

## 用户登录

此功能用于用户通过帐号、手机号或邮箱进行登录。

### API 端点

- **URL:** `/core/login`
- **Method:** `POST`

### 请求参数

| 参数          | 类型   | 描述                   | 示例       | 是否必须 |
| :------------ | :----- | :--------------------- | :--------- | :------- |
| `account`     | string | 登录帐号、手机号、邮箱 | `admin`    | 是       |
| `password`    | string | 登录密码               | `admin888` | 是       |
| `captcha`     | string | 验证码                 |            | 否       |
| `captcha_key` | string | 验证码key              |            | 否       |

### 使用示例 (React & TanStack Query)

您可以使用 `useMutation` 来处理登录请求。

```jsx
import { useMutation } from '@tanstack/react-query'
import { useState } from 'react'

// 定义一个用于登录的异步函数
const loginUser = async (credentials) => {
  const response = await fetch('/core/login', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
    },
    body: JSON.stringify(credentials),
  })

  if (!response.ok) {
    // 在实际应用中，您可能需要解析错误响应体
    const errorData = await response.json()
    throw new Error(errorData.message || '登录失败')
  }

  return response.json()
}

function LoginForm() {
  const [account, setAccount] = useState('')
  const [password, setPassword] = useState('')

  const mutation = useMutation({
    mutationFn: loginUser,
    onSuccess: (data) => {
      // 登录成功，data 中包含 token 和 user 信息
      console.log('登录成功:', data)
      // 在这里处理 token，例如保存到 localStorage 并更新用户状态
    },
    onError: (error) => {
      // 登录失败
      console.error('登录失败:', error.message)
    },
  })

  const handleSubmit = (e) => {
    e.preventDefault()
    mutation.mutate({ account, password })
  }

  return (
    <form onSubmit={handleSubmit}>
      <input
        type="text"
        value={account}
        onChange={(e) => setAccount(e.target.value)}
        placeholder="帐号/手机号/邮箱"
      />
      <input
        type="password"
        value={password}
        onChange={(e) => setPassword(e.target.value)}
        placeholder="密码"
      />
      <button type="submit" disabled={mutation.isPending}>
        {mutation.isPending ? '登录中...' : '登录'}
      </button>
      {mutation.isError && <div>错误: {mutation.error.message}</div>}
    </form>
  )
}

export default LoginForm
```

## 用户注册

此功能用于新用户进行注册。

### API 端点

- **URL:** `/core/register`
- **Method:** `POST`

### 请求参数

| 参数                    | 类型   | 描述     | 最小长度 | 最大长度 | 是否必须 |
| :---------------------- | :----- | :------- | :------- | :------- | :------- |
| `name`                  | string | 用户名   | 3        | 20       | 是       |
| `password`              | string | 密码     | 5        | 20       | 是       |
| `password_confirmation` | string | 确认密码 | 5        | 20       | 是       |

### 使用示例 (React & TanStack Query)

同样，您可以使用 `useMutation` 来处理注册请求。

```jsx
import { useMutation } from '@tanstack/react-query'
import { useState } from 'react'

// 定义一个用于注册的异步函数
const registerUser = async (userInfo) => {
  const response = await fetch('/core/register', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
    },
    body: JSON.stringify(userInfo),
  })

  if (!response.ok) {
    const errorData = await response.json()
    throw new Error(errorData.message || '注册失败')
  }

  return response.json()
}

function RegisterForm() {
  const [name, setName] = useState('')
  const [password, setPassword] = useState('')
  const [passwordConfirmation, setPasswordConfirmation] = useState('')

  const mutation = useMutation({
    mutationFn: registerUser,
    onSuccess: (data) => {
      console.log('注册成功:', data)
      // 注册成功后，通常会返回 token 和 user 信息，可以自动登录
    },
    onError: (error) => {
      console.error('注册失败:', error.message)
    },
  })

  const handleSubmit = (e) => {
    e.preventDefault()
    if (password !== passwordConfirmation) {
      alert('两次输入的密码不一致')
      return
    }
    mutation.mutate({ name, password, password_confirmation: passwordConfirmation })
  }

  return (
    <form onSubmit={handleSubmit}>
      <input
        type="text"
        value={name}
        onChange={(e) => setName(e.target.value)}
        placeholder="用户名"
      />
      <input
        type="password"
        value={password}
        onChange={(e) => setPassword(e.target.value)}
        placeholder="密码"
      />
      <input
        type="password"
        value={passwordConfirmation}
        onChange={(e) => setPasswordConfirmation(e.target.value)}
        placeholder="确认密码"
      />
      <button type="submit" disabled={mutation.isPending}>
        {mutation.isPending ? '注册中...' : '注册'}
      </button>
      {mutation.isError && <div>错误: {mutation.error.message}</div>}
    </form>
  )
}

export default RegisterForm
```

## 用户退出

此功能用于用户退出登录。调用此接口需要有效的认证 `token`。

### API 端点

- **URL:** `/core/logout`
- **Method:** `POST`

### 请求参数

此接口无需请求体参数，但需要在请求头中携带认证 `token`。

### 使用示例 (React & TanStack Query)

```jsx
import { useMutation } from '@tanstack/react-query'

// 假设您有一个函数来获取认证 token
const getAuthToken = () => localStorage.getItem('authToken')

const logoutUser = async () => {
  const token = getAuthToken()
  const response = await fetch('/core/logout', {
    method: 'POST',
    headers: {
      Authorization: `Bearer ${token}`,
    },
  })

  if (!response.ok) {
    throw new Error('退出失败')
  }

  return response.json()
}

function LogoutButton() {
  const mutation = useMutation({
    mutationFn: logoutUser,
    onSuccess: () => {
      console.log('退出成功')
      // 在这里处理退出后的逻辑，例如清除本地 token，跳转到登录页
      localStorage.removeItem('authToken')
      window.location.href = '/login'
    },
    onError: (error) => {
      console.error('退出失败:', error.message)
    },
  })

  return (
    <button onClick={() => mutation.mutate()} disabled={mutation.isPending}>
      {mutation.isPending ? '退出中...' : '退出登录'}
    </button>
  )
}

export default LogoutButton
```

## 找回密码

此功能用于用户通过帐号重置密码。

### API 端点

- **URL:** `/core/findPassword`
- **Method:** `POST`

### 请求参数

| 参数                    | 类型   | 描述               | 是否必须 |
| :---------------------- | :----- | :----------------- | :------- |
| `account`               | string | 需要找回密码的帐号 | 是       |
| `password`              | string | 新密码             | 是       |
| `password_confirmation` | string | 确认新密码         | 是       |

### 使用示例 (React & TanStack Query)

```jsx
import { useMutation } from '@tanstack/react-query'
import { useState } from 'react'

const findPassword = async (data) => {
  const response = await fetch('/core/findPassword', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
    },
    body: JSON.stringify(data),
  })

  if (!response.ok) {
    const errorData = await response.json()
    throw new Error(errorData.message || '密码重置失败')
  }

  return response.json()
}

function FindPasswordForm() {
  const [account, setAccount] = useState('')
  const [password, setPassword] = useState('')
  const [passwordConfirmation, setPasswordConfirmation] = useState('')

  const mutation = useMutation({
    mutationFn: findPassword,
    onSuccess: () => {
      console.log('密码重置成功')
      // 提示用户密码已重置，可以跳转到登录页
    },
    onError: (error) => {
      console.error('密码重置失败:', error.message)
    },
  })

  const handleSubmit = (e) => {
    e.preventDefault()
    if (password !== passwordConfirmation) {
      alert('两次输入的密码不一致')
      return
    }
    mutation.mutate({ account, password, password_confirmation: passwordConfirmation })
  }

  return (
    <form onSubmit={handleSubmit}>
      <input
        type="text"
        value={account}
        onChange={(e) => setAccount(e.target.value)}
        placeholder="请输入您的帐号"
      />
      <input
        type="password"
        value={password}
        onChange={(e) => setPassword(e.target.value)}
        placeholder="新密码"
      />
      <input
        type="password"
        value={passwordConfirmation}
        onChange={(e) => setPasswordConfirmation(e.target.value)}
        placeholder="确认新密码"
      />
      <button type="submit" disabled={mutation.isPending}>
        {mutation.isPending ? '正在处理...' : '重置密码'}
      </button>
      {mutation.isError && <div>错误: {mutation.error.message}</div>}
    </form>
  )
}

export default FindPasswordForm
```
