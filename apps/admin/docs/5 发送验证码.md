# 发送验证码

## 功能描述

发送验证码到指定的邮箱或手机号。

## 接口信息

- **路径**: `/core/sendCode`
- **方法**: `POST`

## 请求参数

| 参数名  | 类型   | 是否必填 | 描述         | 示例值              |
| :------ | :----- | :------- | :----------- | :------------------ |
| account | string | 是       | 邮箱或手机号 | `2300071698@qq.com` |

## 响应结果

### 成功

```json
{
  "success": true,
  "message": "验证码发送成功"
}
```

### 失败

```json
{
  "success": false,
  "message": "发送验证码失败"
}
```

## 使用示例

```typescript
import { useMutation } from '@tanstack/react-query'
import axios from 'axios'

const sendCode = async (account: string) => {
  const { data } = await axios.post('/core/sendCode', { account })
  return data
}

const { mutate } = useMutation({
  mutationFn: sendCode,
  onSuccess: (data) => {
    console.log(data.message)
  },
})

mutate('2300071698@qq.com')
```
